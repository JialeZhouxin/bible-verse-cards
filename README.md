# 圣经金句

> 每日一句神的话语，滋养心灵得平安。

一个轻量的圣经金句抽取应用，帮助信徒每日从神的话语中得着安慰、力量、智慧与盼望。

## 功能

- 随机抽取圣经金句
- 按主题分类筛选（安慰、爱与恩典、信心、力量、智慧、赦免、盼望）
- 按灵修深度筛选（初信、成长、深化）
- 本地保存灵修记录与心得
- 一键复制分享经文

## 使用方式

1. 直接打开 `index.html`。
2. 点击"抽取金句"获取今日灵粮。
3. 可记录心得并查看灵修历史。

## 技术实现

- 纯前端：HTML + CSS + JavaScript
- 本地存储：`localStorage`

## 经文数据来源

精选圣经经文，涵盖：
- **安慰与盼望**：在患难中得着平安
- **爱与恩典**：体验神的爱
- **信心与信靠**：坚定信仰根基
- **力量与勇气**：面对挑战的力量
- **智慧与引导**：生活中的智慧
- **赦免与更新**：罪得赦免的恩典
- **盼望与未来**：永恒的盼望

- `http(s)://` 场景加载模块入口 `src/main.js`
